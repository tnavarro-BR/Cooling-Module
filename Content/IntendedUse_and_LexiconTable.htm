Intended Use

The CFX Opus 96 Dx Real-Time PCR System or CFX Opus 384 Dx Real-Time PCR System with is intended to perform fluorescence-based PCR to detect and quantitate nucleic acid sequences. The systems and software are intended for in vitro diagnostic use by trained laboratory technicians. The systems are intended to be used with third-party diagnostic nucleic acid tests, which have been manufactured and labeled for diagnostic purposes.
